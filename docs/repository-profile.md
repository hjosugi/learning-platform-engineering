# Repository Profile: learning-platform-engineering

Last verified: 2026-06-20

## GitHub Description

Platform engineering learning repo for CI/CD, security, observability, and operations.

## Topics

- `learning`
- `devops`
- `github-actions`
- `opentelemetry`
- `security`

## Audience

Engineers who want to practice the operational layer around applications: CI, containers, observability, security checks, release notes, and runbooks.

## First Milestones

1. Add GitHub Actions templates for Node, Python, Java, and Go.
2. Add Docker and Compose examples for local services.
3. Add an OpenTelemetry primer with trace/metric/log vocabulary.
4. Add an OWASP and secrets checklist.
5. Add release checklist and runbook templates.

## Public Safety Checklist

- No cloud secrets or kubeconfigs.
- No production hostnames or incident details.
- Security notes are educational and do not expose private systems.
- Templates use placeholders and `.env.example`.
