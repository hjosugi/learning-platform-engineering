# Repository Profile: learning-platform-engineering

Last verified: 2026-06-20

## GitHub Description

Platform engineering learning repo for CI/CD, NGINX, security, observability, WebTransport, and operations.

## Topics

- `learning`
- `devops`
- `github-actions`
- `opentelemetry`
- `security`
- `webtransport`

## Audience

Engineers who want to practice the operational layer around applications: CI, containers, reverse proxying, real-time networking, observability, security checks, release notes, and runbooks.

## First Milestones

1. Add GitHub Actions templates for Node, Python, Java, and Go.
2. Add Docker and Compose examples for local services.
3. Add an NGINX reverse proxy example in front of a local API.
4. Add an OpenTelemetry primer with trace/metric/log vocabulary.
5. Add WebSocket/SSE/WebTransport comparison notes.
6. Add an OWASP and secrets checklist.
7. Add release checklist and runbook templates.

## Public Safety Checklist

- No cloud secrets or kubeconfigs.
- No production hostnames or incident details.
- Security notes are educational and do not expose private systems.
- Templates use placeholders and `.env.example`.
