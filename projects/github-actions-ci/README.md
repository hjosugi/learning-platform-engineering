# GitHub Actions CI

Workflow template for learning repository-local checks.

The workflow is stored here as a template instead of `.github/workflows/ci.yml` so it does not unexpectedly run for every experiment.

## Inspect

```bash
python3 projects/github-actions-ci/check_workflow.py
```

## Activate

Copy `ci.yml` to `.github/workflows/ci.yml` in a repo after adjusting the commands.
